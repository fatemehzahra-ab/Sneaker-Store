# 👟 Sneaker Store

This project is a mobile-first sneaker shop that I built using **Vanilla JavaScript**, **HTML**, **Tailwind CSS**, and a provided backend API.

The goal of this project was to build a complete e-commerce experience without using any frontend frameworks. Everything from rendering products and handling user interactions to managing the cart and authentication was implemented with pure JavaScript and DOM manipulation.

## Features

* User signup and login
* Form validation and error handling
* Protected pages for authenticated users
* Product listing with infinite scroll
* Product search with debounce
* Search history
* Product filtering
* Product details page
* Add to cart functionality
* Cart management (increase, decrease, remove items)
* Checkout flow
* Order confirmation
* LocalStorage persistence
* Responsive mobile-first UI

## Technologies

* JavaScript (ES6)
* DOM API
* HTML
* Tailwind CSS
* REST API
* LocalStorage

## What I Learned

Working on this project helped me improve my understanding of:

* DOM manipulation
* Event handling
* Working with APIs
* Authentication flows
* Managing application state without frameworks
* Building multi-page applications
* Creating reusable JavaScript code
* Mobile-first development
